# 🔧 QR Tools Pro

A sleek, modern web application featuring **QR Code Generation** and **Message Encryption** tools with a distinctive black and orange theme.

## 🌟 Features

### 🔍 QR Code Generator
- **Instant Generation**: Convert any text, URL, or message into a QR code
- **Multiple Sizes**: Choose from Small (200px), Medium (300px), or Large (400px)
- **Real-time Preview**: Auto-generates QR codes as you type
- **Download Feature**: Save QR codes as PNG images
- **Clean Interface**: Intuitive design with visual feedback

### 🔐 Message Encrypter/Decrypter
- **Advanced Encryption**: Uses industry-standard AES encryption
- **Bidirectional**: Both encrypt and decrypt messages
- **Custom Keys**: Set your own encryption/decryption passwords
- **One-Click Copy**: Copy encrypted/decrypted text to clipboard
- **Secure Processing**: All encryption happens locally in your browser

## 📱 Usage Instructions

### QR Code Generator
1. **Enter Content**: Type any text, URL, or message in the text area
2. **Select Size**: Choose your preferred QR code dimensions
3. **Generate**: Click "Generate QR Code" or wait for auto-generation
4. **Download**: Click the download button to save the QR code as PNG

### Message Encrypter
1. **Enter Message**: Type the text you want to encrypt/decrypt
2. **Set Key**: Enter a secure password for encryption
3. **Process**: Click "Encrypt" to secure your message or "Decrypt" to reveal it
4. **Copy**: Use the copy button to save the result to clipboard

## 🛡️ Security Features

- **Client-Side Processing**: All encryption happens in your browser
- **No Data Storage**: Messages and keys are never stored or transmitted
- **AES Encryption**: Military-grade encryption standard
- **Secure Random**: Uses cryptographically secure random number generation
